Hii iam vali and this is "Valis AI Chatbot"

This is my first AI chatbot project.

I built this project to learn how a frontend and backend talk to each other
and how to connect an AI model using an API.

The chatbot takes a user message and gives an AI-generated reply,just a simple ai chat bot.
---

## What this project does

- User types a message in the browser
- The message is sent to the backend
- Backend sends the message to Google Gemini AI
- AI reply is sent back and shown on the screen

---

## Technologies used

- React (frontend)
- Node.js and Express (backend)
- LangChain (JavaScript)
- Google Gemini API
